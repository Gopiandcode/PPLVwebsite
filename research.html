UCL PPLV Research Area


The UCL Research Team is leading new developments into this field.
Particularly of note, Peter O'Hearn of the UCL PPLV research team developed Separation Logic which as has facilitated the development of programs to verify programs written in imperative programmming languages. Members of the UCL PPLV research team were also directly involved in developing Facebook's INFER program, implementing separation logic which is used to ensure that phone applications can be developed quickly.

Impacts

As PPLV is centered around improving the logical concepts behind Computer Science, research done by PPLV has impacts over a huge breadth of fields. We've chosen to include a couple impacts that we found the most interesting below. Hopefully this should help you to get an idea of what the PPLV department does.

Gaming Market ( Boosted automatic verification Research Project )

The core code (known as kernel code) for the Graphical Processing Unit (GPU), is responsible for ensuring a high frame rate, smooth gameplay and a good visual appearnce. This code is now being developed by automatic verification algorithims developed by researchers in PPLV.
The kernel code is the low level code required for the GPU to function - previously it was developed and tested by humans, however, thanks to PPLV, it is now being developed using automatic verification algorithms. These algorithms find a way to maxmimise memory usage and identify bugs faster than any human could. These increases in efficiency may be one of the main factors in the UK gaming industry's recent economic growth - there is currently a high demand for GPUs.

Phone industry ( Program Verification Research Project )

Researchers at UCL also helped to develop Separation Logic and Bunched Logic, which are revolutionary new Logical models which allow imperative programs (written in languages such as C, Python, etc.) to be automatically verified. The results of these discoveries are used by FaceBook in their INFER project, which is used to check phone applications for memory leaks and other problems. This facilitates the quick development of code for mobile devices. Currently there are over 250,000,000 android phones alone in use, all running similarly checked software, so it's easy to see how this development has actually shaped the modern world as we know it.

Business Sector ( Algebra and Logic for Policy and Utility in Information Security Project )

By producing a universal way of representing security issues within a system, managers within the industry can more easily model their policies to be sustainable, effective and affordable. For example environmental polciies such as recycling machines and paper and conforming to specfic laws within a country. By using algebra and logic as a foundation to create models this addresses security flaws within organisations in relation to national laws, regulations and agreements.
