What is PPLV?

PPLV is a research area focused on Computational Logic and it's applications to other areas.
Research Projects in this area often either seek to develop new computational logic theories or to use them to create rigorous mathematical models that accurately describe and predict physical systems.

What is Computational Logic?

As PPLV is a research area based around the core concept of Computational Logic, before moving forward, it's probably best to first explain what logic is...


A Brief Introduction to Logic

Note: For the purpose of brevity, this overview is very rough - if you are interested, check the reference for links that can explain this topic in greater detail

Logic, in the mathematical sense is the study of developing rules and operations to describe concepts. In Computer Science, Logic is specifically used to describe Computational Algorithms.

An analogous example which may illustrate the concept more clearly is Boolean Logic and it's relationship with circuits.

In Boolean Logic, values are either True (1) or False (0) and there exist operators (AND, OR, XOR etc.) which return True or False, depending on their input. As these exact operations and values are representable using circuits, we could describe Boolean Logic as a theoretical description of circuits.


Computational Logic encompasses Boolean Logic, but also includes other types of Logic such as Separation Logic and Hoores Logic, which describe computers at a higher level of abstraction (as in describing the programs on the computer rather than the bits that make it up).

Incidentally, one of the core developments in Computational Logic, which is now used throughout the world to verify imperative programs before release, Separation Logic, was developed by a Researcher at UCL's PPLV department - Peter O'Hearn.


So, What is PPLV?

So, In Conclusion PPLV covers a vast area of computers in nearly all fields. This ranges from electrical too mechanical and even biochemical. It focuses on producing logic – the foundations of computer science which are then developed into principles of programming allowing us to communicate and interact with computers in a certain way (different programming languages). The process of verification is then used in order to validate the outcome of such findings. It is a long rigorous process requiring large amount of research, time and skill.
